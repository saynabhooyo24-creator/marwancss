Htmyl

HTML waa laf-dhabarta (structure)
ee bogga internetka. Waxay sheegtaa
waxa boggu ka kooban yahay, sida
cinwaannada, qoraallada, sawirrada,
badhamada, iyo foomamka.

Css

CSS waxay qurxisaa HTML. Waxay
go'aamisaa midabka, cabbirka, fogaanta,
qaabka, iyo sida boggu u ekaanayo.

Javascript

JavaScript waxay bogga ka
dhigtaa firfircoon (interactive). Waxay
ka jawaabtaa ficillada isticmaalaha,
sida gujinta badhan,
buuxinta foom, ama xisaabinta.
Tusaalaha guriga:
HTML = Guriga.
CSS = Qurxinta guriga.
JavaScript = Korontada, albaabka
otomaatiga ah, iyo qalabka shaqaynaya.
